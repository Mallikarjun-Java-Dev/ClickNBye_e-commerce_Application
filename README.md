# ClickNBuy

A full-stack e-commerce web application built using Spring Boot.

## Features

- User Registration/Login
- Product Management
- Shopping Cart
- Order Placement
- Spring Security Authentication
- Razorpay Payment
- Cloudinary Image Upload
- Email Notifications
- Responsive UI

## Tech Stack

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Thymeleaf
- MySQL
- Maven
- Bootstrap
- Docker
